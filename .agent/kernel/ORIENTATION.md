\# Orientation Loop



Status: AGENT LAYER

Parent: .agent/kernel/KERNEL.md



\## Purpose



Convert bootstrap evidence into a repository understanding.



Bootstrap gathers facts.



Orientation explains what those facts mean.



\## Inputs



Required:



\- Bootstrap output

\- Constitution

\- Roadmap

\- Specifications

\- Contracts

\- Schemas

\- Interfaces

\- Workflows

\- Source implementation

\- Tests



\## Orientation Questions



The loop must answer:



1\. What is this repository?



2\. What is the stated mission?



3\. What is the governing authority?



4\. What phases exist?



5\. Which phase is currently implemented?



6\. Which phase is partially implemented?



7\. Which phase is specified but not implemented?



8\. Which phase appears absent?



9\. Which authoritative records exist?



10\. Which authoritative records are missing?



11\. What constitutional requirements remain unsatisfied?



12\. What implementation gaps currently exist?



13\. What assumptions are being made?



14\. What evidence supports those assumptions?



\## Required Outputs



The loop must produce:



Mission



Governance Authority



Implemented Components



Partially Implemented Components



Specified But Not Implemented Components



Missing Components



Constitutional Coverage



Roadmap Coverage



Known Gaps



Evidence Confidence



\## Confidence Rules



Every conclusion must identify:



\- evidence source

\- evidence location

\- confidence level



Unsupported conclusions are prohibited.



\## Drift Detection



The loop must identify:



\- roadmap drift

\- specification drift

\- implementation drift

\- governance drift

\- constitutional drift



Drift must be classified as:



\- NONE

\- POSSIBLE

\- CONFIRMED



\## Output Classification



The loop must assign exactly one repository state:



\- SPECIFICATION\_HEAVY

\- IMPLEMENTATION\_HEAVY

\- BALANCED

\- EARLY\_STAGE

\- MID\_IMPLEMENTATION

\- LATE\_IMPLEMENTATION

\- PRODUCTION\_READY



The classification must be evidence-based.
