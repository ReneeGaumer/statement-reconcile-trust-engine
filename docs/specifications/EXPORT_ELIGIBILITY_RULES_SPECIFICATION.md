# Export Eligibility Rules Specification

Status: AUTHORITATIVE SPECIFICATION
Version: 1.0.0
Constitution Reference: docs\constitution\CONSTITUTION.md
Effective Date: 2026-06-05

## 1. Purpose

Defines eligibility rules governing export release decisions.

## 2. Eligibility Requirements

Exports are eligible only when supported by authoritative records, deterministic rules, evidence lineage, decision ledger entries, and audit package references.

## 3. Ineligibility Conditions

- active export embargo
- failed courtroom test
- unresolved critical exception
- missing evidence lineage
- missing audit package
- unreconstructable trust determination

## 4. Rule Requirements

Eligibility decisions must be deterministic, reproducible, and auditable.

## 5. Completion Standard

Eligibility determination is complete when an independent reviewer can reproduce the release decision.
