# Courtroom Test Specification

Status: AUTHORITATIVE SPECIFICATION
Version: 1.0.0
Constitution Reference: docs\constitution\CONSTITUTION.md
Effective Date: 2026-06-05

## 1. Purpose

The Courtroom Test defines the standard for reconstructing, explaining, and defending trust determinations.

## 2. Required Questions

Every material trust conclusion must answer:

- Can this be trusted?
- Why?
- What evidence supports that conclusion?
- What risks remain?
- How exactly was the conclusion reached?

## 3. Reconstruction Requirements

Each conclusion must identify:

- deterministic rule
- evidence source
- calculation path
- exception influence
- decision rationale

## 4. Failure Rule

If a conclusion cannot be reconstructed, explained, and defended, it is invalid and must generate an exception.

## 5. Completion Standard

The Courtroom Test is satisfied when an independent reviewer can reproduce and defend the conclusion without undocumented assumptions.
