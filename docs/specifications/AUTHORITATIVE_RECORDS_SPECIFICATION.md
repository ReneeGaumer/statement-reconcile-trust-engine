# Authoritative Records Specification

Status: AUTHORITATIVE SPECIFICATION
Version: 1.0.0
Constitution Reference: docs\constitution\CONSTITUTION.md
Effective Date: 2026-06-05

## 1. Purpose

This specification defines the authoritative records of the Trust Assurance Platform.

## 2. Authoritative Records

The authoritative records are:

- Trust Record
- Evidence Lineage
- Decision Ledger
- Audit Package
- Exception Record

Derived artifacts are not authoritative.

## 3. Supremacy

When derived artifacts conflict with authoritative records, authoritative records govern.

## 4. Completion Standard

Authoritative records are complete when trust determinations can be independently reconstructed and defended.
