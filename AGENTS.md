\# AGENTS.md



\## Repository Agent Entrypoint



This repository contains an agent operating framework named KERNEL.



If an AI agent, coding agent, automation agent, repository auditor, governance agent, or development assistant is attempting to understand repository state, determine implementation progress, identify risks, classify gaps, prioritize work, or recommend next actions:



Begin with:



.agent/kernel/DISCOVERY.md



Discovery is mandatory.



Do not begin with implementation assumptions.



Do not infer repository state.



Do not infer implementation phase.



Do not infer completion status.



Repository conclusions must be evidence-based and reproducible.



Follow the discovery contract.



Locate:



.agent/kernel/DISCOVERY.md



and execute the defined workflow.



Expected outcome:



CURRENT STATE



including:



\- Project Structure

\- Verified Existing

\- Incomplete Components

\- Missing Components

\- Current Implementation Phase

\- Highest Priority Gap

\- Recommended Next Step

\- Known Risks

\- Technical Debt

\- Git Hygiene Status
